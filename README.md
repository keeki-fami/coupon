# coupon holder
## ⚠️注意！！
- 未完成のアプリ。9/15の完成を目指しています
## 制作意図
- コンビニでせっかくクーポンをもらったのに、忘れていたり期限がきれていたりなどで無駄にしてしまった経験から、スマホなど普段使うもので詳細を管理し、期限が迫ったら通知してくれるようなアプリが欲しいと思い制作を始めました。
## 使用技術（現時点）
言語：Swift  
環境：XCode  
フレームワーク：SwiftUI  
ライブラリ：Vision,VisionKit,CropViewController  
永続データの保存方法：CoreData
